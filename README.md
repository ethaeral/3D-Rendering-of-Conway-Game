# 3D Rendering of Conway's Game of Life

## Demo
https://rainbowoflife.vercel.app/

## Features
Enable rotate animation<br/>
Automate game of life rules<br/>
Enable outlines<br/>
Customize cube size

## Get Started
Download the repo. Cd into the folder and run these scripts.
```
npm i 
npm start
```
